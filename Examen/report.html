Primer Nivel: Caracterización de Dominio

Hoy en día, la industria de los videojuegos ha crecido enormemente, resultando en uno de los hobbies más comunes en el mundo. El catálogo más diverso actualmente corresponde al de los juegos de computador, surgiendo así una comunidad inmensa de personas apasionadas por este gran arte. Estas personas corresponden al usuario objetivo de este trabajo, el que busca entregar funcionalidades de utilidad para realizar consultas sobre el catálogo de la plataforma/tienda más popular en la actualidad, conocida como Steam. Para realizar esto se decidió utilizar un conjunto de datasets que contienen información sobre los videojuegos de Steam, aplicando una fase de preprocesamiento para unificarlos en un sólo dataset de interés.

Consideraciones:

  • Los datasets originales se pueden encontrar en el siguiente link: https://www.kaggle.com/nikdavis/steam-store-games.
  • El dataset unificado es el que se encuentra en la carpeta 'data' entregada, y es el utilizado por la herramienta de visualización.
  • Para realizar el preprocesamiento se utilizó la librería pandas de python (con python 3.8), uniendo los datos relevantes de todos los videojuegos en un sólo dataset mediante joins entre los datasets originales. El script utilizado se encuentra adjunto ('process.py').
  • La herramienta está hecha en Inglés, buscando apelar a una mayor cantidad de usuarios a nivel global.
  • Para poder aprovechar la herramienta al 100% se recomienda estar conectado a internet (por temas de imágenes customizadas).
